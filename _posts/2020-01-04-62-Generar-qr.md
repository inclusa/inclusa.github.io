Instal·lem la utilitat:

```bash
sudo apt install qrencode
```

Generem la imatge a partir d'una url:

```bash
qrencode -m 5 -o qrcode.png 'https://inclusa.github.io'
```

Obtenim:

![inclusa.png](../img/inclusa.png)

Visualitzem la imatge:

```bash
feh inclusa.png
```
